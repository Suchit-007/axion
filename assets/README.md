# Assets

This folder contains media assets for the project documentation.

## Files:

- **demo-video.mp4** - 2-3 minute demonstration video showing all features

## How to add the demo video:

1. Record a screen capture of the application showing:

   - Login flow
   - Incident reporting process
   - Form auto-detection features
   - Dashboard and statistics
   - Heatmap visualization
   - Incidents list and details
   - User navigation

2. Edit the video (optional):

   - Add title cards
   - Show feature highlights
   - Include narration or background music
   - Keep it under 3 minutes

3. Export as MP4 (H.264 codec, 1920x1080 or 1280x720)

4. Save as `demo-video.mp4` in this folder

5. Commit and push to repository

## Tools for screen recording:

- OBS Studio (Free, Open Source)
- ScreenFlow (Mac)
- Camtasia (Professional)
- ShareX (Windows, Free)
- ffmpeg (Command line)
